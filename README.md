# Playground

A collection of small apps and experiments built with AI assistance.

---

## Projects

### Screensavers

A retro-inspired screensaver control panel for Windows, modeled after the classic *After Dark* screensaver suite. Built with Electron, it lets you launch multiple animated screensaver modules from a single launcher.

**Stack:** Electron, React (via CDN), WebGL / Canvas

**Run locally:**
```bash
cd screensavers
npm install
npm start
```

**Build a portable `.exe`:**
```bash
npm run build
```

---

### Elemental Wallpapers

Animated WebGL shader wallpapers that run in the browser. Each wallpaper is a self-contained GLSL shader producing real-time procedural visuals.

**Stack:** HTML/CSS, WebGL (GLSL shaders)

**Run locally:** Open `ElementalWallpapers/index.html` in any modern browser — no build step required.
